# 鋳造価格 計算ツール

ワックス原型の重量から、**シルバー・真鍮**などの金属の鋳造価格を見積もる WEB ツールです。
ジュエリー制作の見積り用。ビルド不要の単一 HTML ファイル（`index.html`）で動きます。

## 使い方

1. `index.html` をブラウザで開く
2. **ワックス重量(g)** を入力
3. 金属（シルバー / 真鍮）を選択
4. 工賃・地金代・税込合計が自動計算されます

地金相場は日々変動するので、`⚙ 単価・係数の設定` から **地金単価** を更新できます。
変更した値はブラウザに保存されます（`初期値に戻す` でホワイトボードの値に戻せます）。

## 計算式

ホワイトボードの計算ロジックをそのまま実装しています。

```
金属重量 = ワックス重量 × 比重 + 湯口分(2g)
工賃     = 金属重量 × 工賃単価(円/g)
地金代   = 金属重量 × 地金単価(円/g)
合計     = (工賃 + 地金代) × (1 + 税率)
```

### 初期値（2025/5/25 時点）

| 金属 | 比重 | 工賃 (円/g) | 地金単価 (円/g) |
|------|------|------------|----------------|
| シルバー (SV) | 10.5 | 132 | 549 |
| 真鍮 | 8.6 | 154 | 33 |

※ 消費税 10% / 湯口・鋳造追加重量 +2g

## 公開（GitHub Pages）

このリポジトリを GitHub Pages で公開すると、スマホからもアクセスできます。

1. GitHub の **Settings → Pages**
2. Source を `Deploy from a branch`、ブランチを選択して `/ (root)` を指定
3. 表示された URL で `index.html` が公開されます
