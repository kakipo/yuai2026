## 鋳造価格 計算ツール（`/yuai2026/`）

ワックス原型の重量から、**シルバー・真鍮**などの金属の鋳造価格を見積もる WEB ツールです。
ジュエリー制作の見積り用。単一 HTML ファイル（`yuai2026/index.html`）で動きます。

### 使い方

1. `tools.kakipo.com/yuai2026/` を開く（ローカルなら `yuai2026/index.html` をブラウザで開く）
2. **ワックス重量(g)** を入力
3. 金属（シルバー / 真鍮）を選択
4. 工賃・地金代・合計（税込）が自動計算されます

単価はすべて**税込**で扱い、合計も税込で表示します（消費税を別途加算しません）。
`⚙ 単価・係数の設定` から各単価を手動調整でき、変更はブラウザに保存されます（`初期値に戻す` で既定値に戻せます）。

#### 地金相場の自動取得

シルバーの地金単価は、Seaforce（加工net）の[鋳造料金表](https://www.kakounet.com/cast/cast/price.html)から
**GitHub Actions が日次で取得**（`scripts/update_prices.rb`）し、`yuai2026/prices.json` に保存します。ページ表示時にこれを読み込み、
シルバー地金単価へ自動反映します（取得失敗時は既定値で動作）。真鍮・工賃は固定値のため対象外です。

> 表示価格はあくまで目安です。実際の請求は仕上がり時の地金相場によります（Seaforce の注記に準拠）。

### 計算式

```
金属重量 = ワックス重量 × 比重 × (1 + 湯口率)   ※湯口率は金属重量に加算する目減り割合
合計（税込）= 金属重量 × 工賃単価(円/g) ＋ 金属重量 × 地金単価(円/g)
```

#### 既定値（税込）

| 金属 | 比重 | 工賃 (円/g) | 地金単価 (円/g) |
|------|------|------------|----------------|
| シルバー (SV925/950) | 10.5 | 132 | 476 ※自動更新 |
| 真鍮 | 8.6 | 154 | 33 |

※ 湯口・鋳造追加率 +2%（金属重量に対して） / 地金単価は税込

## 公開のしくみ（GitHub Pages + 独自ドメイン）

- `main` への push で `.github/workflows/deploy-pages.yml` が走り、リポジトリ全体を GitHub Pages にデプロイします。
- ルートの `CNAME` ファイルで独自ドメイン `tools.kakipo.com` を指定しています。
- 新しいツールを追加するときは、リポジトリ直下に `<ツール名>/index.html` を置くだけで `tools.kakipo.com/<ツール名>/` で公開されます。
- `.github/workflows/update-prices.yml` が日次（cron）で `scripts/update_prices.rb`（Ruby）を実行し、地金相場を取得して `yuai2026/prices.json` を更新・再デプロイします（手動実行も可）。

### 独自ドメインの DNS 設定（初回のみ）

DNS プロバイダで以下の CNAME レコードを追加します。

```
tools.kakipo.com.  CNAME  kakipo.github.io.
```

その後、**Settings → Pages → Custom domain** に `tools.kakipo.com` を設定し、証明書が発行されたら **Enforce HTTPS** を有効化します。
